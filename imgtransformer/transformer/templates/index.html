Welcome to the simple Image Transformation API

General information

Currently it supports 3 operations:

  1. "/api/rotate/DIRECTION/DEGREE/" - rotates image by custom angle
    DIRECTION - cw for clockwise or ccw for counterclockwise
    DEGREE - positive integer from 0 to 999

  2. "/api/resize/PERCENTAGE/" - resizes image to some percent of its original size
    PERCENTAGE - positive integer from 1 to 999
  3. "/api/bw/" - to convert colour space of the image from RGB to Black and White

How to use it

  1. Choose necessary endpoint
  2. Specify modification parameters in URL
  3. Select image file to be modified and upload it
  4. You will receive modified file in response

Some usage examples

  • /api/rotate/cw/90/ - rotate image by 90° clockwise
  • /api/rotate/ccw/43/ - rotate image by 43° counterclockwise
  • /api/resize/90 - resize image to 90% of its orifinal size
  • /api/bw/ - colourspace convertion
